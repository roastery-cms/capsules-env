# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.0.5]

### Added

- JSDoc documentation for the `baristaEnv` plugin function, covering its type
  parameter, arguments, return type, thrown exception, and a usage example.
- JSDoc documentation for the `ServerDependenciesDTO` schema and its inferred
  type, including field descriptions for `PORT` and `NODE_ENV`.
